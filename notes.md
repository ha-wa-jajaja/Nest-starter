### Creating a module

```shell
nest g module {module name}
```

### Adding controller to a module

```shell
nest g controller {module name}
```

### Init Prisma

```
npx prisma init
```

### Prisma migration

```
npx prisma migrate {flag} --name {name}
```

e.g.

```
npx prisma migrate dev --name init
```

### Prisma generate

`npx prisma generate`
